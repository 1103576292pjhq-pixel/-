# MXFP8 格式说明（项目版）

## 1. 数据组织
- 一个 `block` 包含 `32` 个 `E4M3` 元素
- 每个 block 共享 `1` 个 `E8M0 scale`
- `A/B` 以 block 为单位进入 `dot32` 计算

## 2. 项目内部计算主线
- 元素部分先做 `E4M3 x E4M3`
- 块级 `scale` 不直接扩到每个乘法 lane，而是后置到指数路径
- 列级得到 `dot32` 后，再进入 `FP32` 累加器

## 3. 当前参考模型约定
- `E4M3`：
  - 零值：低 7 位全零
  - 当前项目参考实现把 `exp=0xf,mant=0x7` 视为 NaN
  - subnormal 走单独解码路径
- `E8M0`：
  - 当前项目参考实现把 `0xff` 视为 NaN
  - 其余编码按 `enc - 127` 转成无偏指数
